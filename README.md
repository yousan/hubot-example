## Installation

In hubot project repo, run:

`npm install hubot-example --save`

Then add **hubot-example** to your `external-scripts.json`:

```json
[
  "hubot-example"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
